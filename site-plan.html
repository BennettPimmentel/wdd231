Healthy Fruit Bakery — Site Plan

Site Name

Healthy Fruit Bakery — A bakery focused on healthy fruit-flavored breads.

Purpose

This site will showcase healthy fruit breads, provide product information, share recipes, and allow users to place breakfast orders.

Target Audience Scenarios

  • María (28): Wants to order breakfast for next morning.
  • Luis (34): Wants to find high-protein fruit breads.

Color Schema

Primary colors:

  • Peach Orange (#FFAD60) — headings and buttons
  • Soft Green (#6A8E7F) — section backgrounds
  • Dark Gray (#333333) — main text
  • White (#FFFFFF) — backgrounds

Typography

The site uses the Montserrat font family. Headings use Montserrat Bold; body text uses Montserrat Regular.
